# 🔮 Korra Smart-Asset Oracle (K-SAO)
> **نظام إدارة وتدقيق الأصول الهندسي الذكي - الإصدار الاستراتيجي**

نظام متطور مصمم خصيصاً لشركة **Korra Energi** لتحويل جداول الإكسيل التقليدية إلى قاعدة بيانات تفاعلية ذكية تعتمد على خوارزميات البحث الدلالي والتصنيف التلقائي.



## 🚀 المميزات الرئيسية (Key Features)
* **البحث الدلالي (Semantic Search):** محرك بحث ذكي يتجاوز الأخطاء الإملائية ويجد الخامات بناءً على "المعنى" والكلمات المفتاحية وليس فقط الحروف المطابقة.
* **التصنيف الهندسي الآلي (Auto-Classification):** يقوم النظام بقراءة وصف الخامة وتصنيفها تلقائياً إلى (كهرباء، ميكانيكا، مدني، أدوات) لتسهيل استخراج التقارير.
* **لوحة تحكم تفاعلية (BI Dashboard):** رسوم بيانية حية توضح توزيع المخزون، كثافة الأصناف في كل مشروع، وحالة جودة البيانات.
* **تصدير التقارير:** إمكانية فلترة آلاف السجلات وتحميلها في ملف إكسيل نظيف ومنظم بضغطة زر.

## 🛠️ الأدوات المستخدمة (Tech Stack)
* **Python:** اللغة الأساسية لبناء المنطق البرمجي.
* **Streamlit:** لإدارة واجهة المستخدم التفاعلية.
* **Pandas:** لمعالجة وتحليل البيانات الضخمة (Data Crunching).
* **Plotly:** لإنشاء الرسوم البيانية الديناميكية.
* **FuzzyWuzzy:** لتنفيذ خوارزميات البحث التقاربي (Fuzzy Logic).
